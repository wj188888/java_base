public class String {
    // 双亲委派机制;安全
    // 1.APP--> EXC ---> BOOT(最终执行)
    // BOOT,优先级最高
    // EXC，其次
    // APP，最后才轮到他去调用

}

## 沙箱安全机制
 组成沙箱的基本组件：
 1. 第一步：字节码校验，核心类是不用进行字节码校验；
 2. 类加载器。防止恶意代码去干涉善意的代码。
 - 从最内层JVM自带的类加载器加载，外层恶意代码无法修改加载顺序；

3. java中将执行程序分为本地代码和远程代码两种吗，本地代码默认是可信任的，而远程代码则不是受信任的
